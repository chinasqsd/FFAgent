# 收件箱 README

> 本文件常驻不删。收件箱是外部抓取/投递内容的入口，处理完成后必须保持只有本文件。

## 投递约定

前端系统（如 DouNote）自动投递内容到本目录。投递内容分两类（frontmatter `category` 字段）：

- **金融**：盘评/研报/产业分析，需要蒸馏后归并入产业卡片
- **AI/其他**：学习笔记类，直接归档

## 处理流程

用户说"处理收件箱"时：

1. 先读本文件
2. 处理 `_收件箱/` 中除 README 外的所有文件
3. **金融类**：核实股票代码 → 蒸馏内容归入对应产业卡片演进日志 → 原文归档
4. **AI类**：直接归档到笔记目录
5. 处理完从 `_收件箱/` 移走，只保留 README
