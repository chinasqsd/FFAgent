## 调取规则

- 分析某只股票 → 读对应个股调研 + 所属产业卡片
- 分析持仓结构 → 读持仓文件 + 调仓计划
- 分析产业机会 → 读产业卡片 + 相关个股调研
- 看资金持续性 → 读资金流向日志 + 当日资金快照
- 重大交易决策 → 触发智囊团评审
- 新资料处理 → 读收件箱处理流程
