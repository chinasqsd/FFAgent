# 个股深度调研逻辑模板

> 用途：给 Agent 集群做单只股票深度拆解时使用。核心不是照搬天孚结论，而是复用它的思考顺序：**产业链位置 → 盈利质量 → 客户/技术风险 → 竞争格局 → 估值定价 → 交易条件 → 证伪条件**。

## 一、模板适用场景

适合：
- 对单只 A 股做首次深度调研；
- 对持仓股做逻辑复核；
- 对热门赛道中的公司判断"是真稀缺，还是蹭概念"；
- 给 Agent 集群分工时，作为详尽报告的一部分。

不适合：
- 盘中临时追涨决策；
- 只看技术面做短线；
- 没有足够公开资料的小票；
- 港股/美股代码未由用户提供的标的。

## 二、核心分析链路

### 1. 先定位：它到底在产业链哪一层赚钱？

不要从"跌了多少、涨了多少"开始。先回答：
- 这家公司卖什么产品，技术路线是什么？
- 它在产业链的上游/中游/下游？上下游各是谁？
- 它的客户是谁？客户集中度、议价能力。
- 它的竞争优势（成本/技术/渠道/品牌/认证壁垒）是什么？

### 2. 盈利质量分析

- 营收增长趋势（3年）——是量的增长还是价格驱动？
- 毛利率趋势——是否在改善？天花板在哪？
- 净利率——费用率结构（研发/销售/管理）。
- ROE——杠杆还是真盈利？
- 现金流——经营现金流能否覆盖投资支出？
- 应收账款和存货周转——有没有被客户占款或积压？

### 3. 风险排查（反面看）

- 客户集中度：前五大客户占营收比例。
- 技术路线风险：有没有替代技术可能淘汰它？
- 大股东/管理层：有没有减持、质押、诉讼？
- 行业周期：当前是上升期还是下行期？
- 监管风险：有没有政策变数。

### 4. 竞争格局

- 国内/全球市场份额。
- 主要竞争对手及相对优劣势。
- 行业壁垒（认证周期/客户粘性/专利壁垒）。
- 新进入者威胁。

### 5. 估值定价

- 当前 PE/PB/PS 在历史分位。
- 与可比公司对比（国内外同行）。
- 如果按 DCF 或分部估值，合理区间在哪。
- 当前价格包含了什么样的市场预期（乐观/中性/悲观）。

### 6. 交易条件

- 流动性：日均成交额、换手率。
- 技术面位置：距MA20/MA60乖离、MACD、RSI。
- 资金面：主力净额趋势、北向资金动向。
- 催化剂：近期有什么财报/产品/政策事件。

### 7. 证伪条件

明确写——什么情况下这个投资逻辑就不成立了？
- 如果XX数据不及预期，目标价下调到多少。
- 如果XX技术路线被替代，这只票就变成"传统公司"。
- 如果大盘破位走熊，它会跌到什么位置。

## 三、输出格式建议

用 Markdown 格式输出，每段开头加 📍 标签区分分析类型：
- 📍 产业链定位
- 📍 盈利质量
- 📍 风险排查
- 📍 竞争格局
- 📍 估值
- 📍 交易条件
- 📍 证伪条件

文末加一个「简单一句话」和一个「如果只能看一个数据/指标」。
