# 飞书通知配置

> ⚠️ 此 URL 是凭证，勿公开分享/提交到公共仓库。

## Webhook 地址

```
{{FEISHU_WEBHOOK_URL}}
```

## 使用约定

- 每完成一件较长/重要的任务（研报、批量蒸馏、评审、情报刷新、落盘等），完成后主动向此 Webhook 发一条简短中文通知（做了什么 + 关键结论/落盘位置）。
- 地址在 `.env` 中配置，各脚本通过环境变量读取。
- 琐碎小回复（一句话确认、纠错等）不必推送，避免刷屏。

## 发送方法（PowerShell，UTF-8）

```powershell
$url = '{{FEISHU_WEBHOOK_URL}}'
$body = @{ msg_type='text'; content=@{ text='通知内容' } } | ConvertTo-Json -Depth 4
Invoke-RestMethod -Uri $url -Method Post -ContentType 'application/json; charset=utf-8' -Body $body
```

返回 `{"code":0,"msg":"success",...}` 即成功。
