# 钱脉系统交接文档

> 2026-07-03 整理，供AI交接使用。

## 一、钱脉是什么

「钱脉系统」是A股资金流向采集+分析系统，2026-06-25上线。核心目的：**判断资金动向的持续性**——哪些行业是"连续流入的真主线"vs"单日脉冲的一日游"。补行情软件只能看当天+近5天的短板。

## 二、系统架构（两个AI的分工）

**小白**（腾讯云服务器上的AI助手）
- 角色：采集端、历史记账员
- 做什么：每交易日15:30后定时跑一次，用thsdk同花顺API采集当日A股资金流向数据
- 产出：JSON文件，包含行业TOP10、固定14个行业、8个概念板块、35只个股

**克劳德**（本助手，AI Agent）
- 角色：消费端、趋势分析师
- 做什么：①盘中实时查个股资金（thsdk现拉）②盘后拉取小白的历史数据 ③累积成时间序列、判断持续性

## 三、数据采集内容

- 固定核心概念板块8个：CPO光模块、HBM高带宽存储、国产算力、PCB、半导体设备、先进封装、存储芯片、人形机器人

### 核心字段

- `amount_wanyi`：两市成交额（万亿）

## 四、数据存储位置

### 小白侧（腾讯云）
- API地址：`https://ai.sgcs.vip/fund_flow/api`
- Token：`fund_flow_2026`
- 调用方式：`?latest=true`（最新交易日）或 `?date=YYYY-MM-DD`（指定日期）

### 克劳德侧（本地）
- 原始数据：`数据/资金流向/YYYY-MM-DD.json`（每日一个文件）
- 采集日志：`数据/资金流向/_fetch.log`
- 分析日志：`产业逻辑/_行业资金流向日志.md`

## 五、定时任务配置

### Windows计划任务
- 任务名：`ETF_QianMai_Fetch`
- 触发器：每交易日16:00（给小白15:30留30分钟余量）
- 动作：`pythonw 工具\fetch_fund_flow.py --cron`（静默模式）

### 手动拉取
```bash
python 工具/fetch_fund_flow.py           # 取最新交易日
python 工具/fetch_fund_flow.py 2026-07-02  # 取指定日期
python 工具/fetch_fund_flow.py --cron     # 静默模式（定时任务用）
```

## 六、盘中实时查询（thsdk）
```bash
python 工具/fund_flow.py 300476 300394 002169 603501 603986  # 查指定票
python 工具/fund_flow.py --pool                               # 查预设监控池35只
```
输出：主力净流入/主力净量/5日涨幅/量比/换手。

## 七、数据分析流程
1. **拉数据**：每交易日盘后跑 `fetch_fund_flow.py`，JSON留档
2. **写分析**：追加到 `产业逻辑/_行业资金流向日志.md`
3. **趋势分析**：每周/关键节点做5日/20日时间序列分析，看资金流转路径
