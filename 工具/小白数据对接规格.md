# 钱脉系统 — 小白 → AI Agent 资金流向数据对接规格

> **系统名**：「钱脉系统」（小白运行在腾讯云的线上资金流向采集系统，2026-06-25 命名）。小白=采集端，AI Agent=消费/分析端。

> **用途**：小白每个交易日定时采集 A股资金流向，存储并提供给 AI Agent 查询；AI Agent 累积成时间序列、判断主线持续性。

## 一、采集时点 & 数据源
- **时点**：每个交易日收盘后定时跑一次（建议 **15:30 后**，数据稳定）。非交易日不跑。
- **数据源**：thsdk 同花顺板块级 API(`market_data_block`) + 个股资金接口。

## 二、采集内容

### A. 行业资金流向（流入 / 流出 各 TOP10）
| key | 含义 |
|---|---|
| `rank` | 排名 1-10 |
| `name` | 板块名 |
| `net_yi` | 主力净额(亿)，流入为正、流出为负 |
| `chg` | 当日涨幅(%) |
| `chg5d` | **5日涨幅(%)——判持续性命门，必带** |
| `vol_ratio` | 量比 |

### B. 个股资金流向（固定清单 35 只，★为核心字段）
| key | 含义 |
|---|---|
| `code` / `name` | 代码 / 名称 |
| `price` / `chg` | 收盘价 / 涨幅(%) |
| ★ `main_net_yi` | **当日主力净额(亿)** |
| ★ `main_net_5d_yi` | **5日主力净额(亿)——判持续性命门** |
| ★ `super_large_yi` | 超大单净额(亿) |
| `turnover` / `vol_ratio` | 换手(%) / 量比 |

### A2. 固定核心行业 & 概念板块（每日全采，不受 TOP10 限制）
**固定核心行业（14个）**：半导体、元件、光学光电子、消费电子、电子化学品、计算机设备、通信设备、其他电子、自动化设备、电池、贵金属、证券、电力、工业金属

**固定核心概念板块（8个）**：CPO光模块、HBM高带宽存储、国产算力（AI芯片）、PCB、半导体设备、先进封装、存储芯片、人形机器人

**个股清单（35只）** 同 `钱脉系统交接文档.md` 所述。

### C. 大盘（附带）
`上证`收盘 / `创业板`收盘 / `成交额万亿`。

## 三、输出 JSON 格式
```json
{
  "date": "2026-06-25",
  "market": {"sh": 4120.28, "cyb": 4371.99, "amount_wanyi": 3.0},
  "industry_in_top10": [{"rank":1,"name":"元件","net_yi":110.18,"chg":3.14,"chg5d":3.52,"vol_ratio":1.00}],
  "industry_out_top10": [{"rank":1,"name":"电池","net_yi":-47.87,"chg":-1.84,"chg5d":-3.17,"vol_ratio":0.93}],
  "industry_core": [{"name":"半导体","net_yi":56.30,"chg":2.21,"chg5d":10.10,"vol_ratio":1.10}],
  "concept_core": [{"name":"CPO光模块","net_yi":42.5,"chg":4.8,"chg5d":12.3,"vol_ratio":1.6}],
  "stocks": [{"code":"300394","name":"天孚通信","price":341.97,"chg":5.13,
               "main_net_yi":-1.5,"main_net_5d_yi":-6.2,"super_large_yi":-2.0,
               "turnover":5.58,"vol_ratio":1.5}]
}
```
单位统一：金额一律「亿元」、比例一律「%」(只填数字、不带符号)。缺失值填 `null`。

## 四、传输方式
**方案A（推荐）**：小白每天把 JSON 文件同步落到用户本地库。
**方案B（备选）**：小白在腾讯云提供 HTTP API。

## 五、分析流程
1. 每天取当日 JSON；
2. 追加进 `产业逻辑/_行业资金流向日志.md`；
3. 定期做趋势分析：①连续N日流入的行业=资金真加仓的真主线 ②忽进忽出=轮动脉冲不追。
