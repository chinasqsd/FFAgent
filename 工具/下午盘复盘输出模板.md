# 下午盘复盘输出模板

> 用途：用户要求"下午盘总结 / 盘后判断 / 明天怎么看 / 持仓股给指引"时，默认按本模板输出。

## 输出顺序

### 1. 一句话定性

先用一句话说明今天属于哪类盘面：缩量阴跌 / 恐慌杀跌 / 分化修复 / 板块共振修复 / 弱反弹 / 真止血。

必须区分"修复苗头"和"转折确认"。未满足资金转正和次日延续前，不把单日翻红说成趋势反转。

### 2. 大盘与市场状态

只写影响操作的事实：
- 指数涨跌、成交是否缩量、涨跌家数。
- 市场是恐慌、观望、修复，还是放量进攻。
- 如果有外部锚（如外盘、关键财报），说明它影响的是哪条链。

### 3. 板块分组对比

按"组"讲，不只逐个报个股：
- 光模块/海外链：中际、新易盛、天孚、东山等是否共振。
- PCB：胜宏、东山、鹏鼎等内部强弱。
- 存储/半导体设备/国产链：兆易、海光、中芯、寒武纪等是否成为资金锚。
- 低位轮动：机器人、商业航天、黄金、创新药等只按轮动处理。

每组必须同时看：
- 价格是否翻红。
- 主力净额是否转正或明显收窄。
- 是否有同组共振。

### 4. 持仓股逐只分析

每只持仓固定四句话：
1. 当前数据：价格、涨跌、主力净额、日内关键位。
2. 今日资金变化：对比上一轮（10分钟或昨日）主力净额变化。
3. 当前位置：距MA20/MA60乖离、在关键位上方还是下方。
4. 操作指引：持有的理由/减仓的条件/止损的距离。

### 5. 明天核心判断与操作策略

明确写：
- 什么情况可以出手/加仓（条件触发式，不靠感觉）。
- 什么情况要减仓撤退。
- 最需要关注的1-2只票和1-2个大盘信号。

### 6. 模板结尾

"以上是收盘后基于今日数据做的情景推演。明天开盘先看大盘闸和资金前排方向，确认信号再动手。"
