# 板块乖离扫盘系统 · 完整说明

> 2026-06-18 建。目的：把"AI硬件主线 + 特别关注复进票"做成每天盘中自动推送的出手雷达，用乖离率(BIAS)分板块自适应⭐告诉用户"能买吗/要卖吗"，治用户"着急追高/砍早/踏空"。
> **本文档是这套系统的唯一总说明，回头要改先读这里。**

## 一、一句话原理
对每只票算 **BIAS乖离率 =(收盘−MA5)/MA5×100%**，再换算成**相对它自己(或本板块)历史常态的 z-score**，映射成**能买★/要卖★**（5★该操作、1★不动）。核心结论来自乖离率研究：①没有固定乖离阈值，随板块波动伸缩；②高乖离会不会回落**取决于动能**（强动能高乖离=别减防踏空，动能转弱才该减）；③超跌(负乖离)反弹最稳。

## 三、数据源（重要）
**全部走腾讯网络接口，不是本地通达信**，所以不受"通达信开盘没当天数据"限制。
- 历史前复权：`qfq_day`（`web.ifzq.gtimg.cn/.../fqkline`）。
- 当前价：`realtime_price`（`qt.gtimg.cn`），**盘中=实时成交价、盘后=收盘价**。
- `cur()` 用"实时价拼到前复权历史(剔除当日bar防重复)"，**盘中任意时刻跑都准**。

## 四、核心设计（都在 sector_bias_radar.py）
**标定（per-sector vs per-stock）**：
- **主线4组**：组内5只**池化**算一个共享 μ/σ（同质板块）。
- **特别关注**：**每只用自己120日历史算自己的 μ/σ**（互不相干，不能共享标尺）。

**z-score → ⭐ 档位**（`buy_n`/`sell_n`/`temp` 函数）：
| z(偏离自己常态) | 温度 | 能买★ |
|---|---|---|
| <−1.5σ | 超跌 | ★★★★★ |
| −1.5~−0.5σ | 偏冷 | ★★★★ |
| −0.5~+0.5σ | 中性 | ★★★ |
| +0.5~+1.5σ | 偏热 | ★★ |
| >+1.5σ | 过热/极热 | ★ |

**要卖★（动能闸门防踏空，核心）**：
- 不贵(z<0.5) → ★（别卖）
- 贵 + 动能还在升(mom5>0) → ★（极热z>2给★★，只是"很贵了"提示，**不强减=防踏空**）
- 贵 + 动能转弱(mom5<0) → ★★★★(偏贵)/★★★★★(极贵) ← **真减仓信号**
> mom5=近5日涨幅(动能)、mom22=近22日涨幅(趋势强度)。**贵≠该卖，必须"贵+涨不动了"才给高卖星**——这是治"乖离一高就砍早"的关键。

## 六、建仓/加仓规则
- **门槛**：能买★★★(中性/回踩到位)=建仓起手；★★★★(偏冷)或趋势确认=加仓；★★/★=不建仓。
- **节奏**：每只3批——首仓★★★下1/3 → ★★★★或企稳+1/3 → 放量站稳+1/3。
- **仓位**：进攻总仓≤12万(留一半≈13万现金，红线)；≤3只，单票6–8万；首仓试仓≈2万。
- **治着急5刹车**：⭐门槛挡手痒/首仓只1/3/加仓需新触发/留一半现金/下单即设止损。

## 七、飞书推送
- **扫盘专用群webhook**：硬编码在 `FEISHU_URL`（见飞书通知配置.md）。
- **推送格式**（`build_feishu`）：顶部「大盘闸」(🟢🟡🟠🔴🔴 bgate) → 「📌今日行动」→ 4主线板块 → 特别关注组 → 「📋节奏」→ 底部图例。
- **手动推**：`python scan_push.py` 或 `python 工具/sector_bias_radar.py --feishu`。
- **盘中自动推**：Windows计划任务 `ETF_Scan_Push`。

## 七·B、大盘 A/B 闸自动警报
推送顶部「📌今日行动」之上，自动加一行**大盘闸**状态，到 B 闸自动红字警报。
- **函数**：`bgate()`
- **判定**（四档）：
  | 状态 | 条件 | 含义 |
  |---|---|---|
  | 🔴🔴 B闸触发 | 上证<下沿 **且**(量比≥1.5**或**≥2只科技ETF破MA20) | 持币/持仓科技腿启动防守 |
  | 🟠 逼近B闸 | 上证<下沿 **或** ≥2只科技ETF破MA20 | 警戒、不出手 |
  | 🟡 破中轨 | 上证<中轨 | A闸仍开但走弱 |
  | 🟢 A闸健康 | 上证≥中轨 | 缩量回踩可出手 |

## 八、常见修改怎么做
| 想改什么 | 怎么改 |
|---|---|
| **加/删标的** | 改 `sector_bias_radar.py` 的 `GROUPS`(主线)或`SPECIAL`(特别关注) + `NAMES`。**⚠️代码必须先核验**。 |
| **改⭐档位松紧** | 改 `buy_n`/`sell_n` 的 z 阈值。 |
| **改标定窗口** | 改 `LOOKBACK`（默认120交易日）。 |
| **改B闸通道值/放量阈值** | 改脚本顶部 `LOWER_RAIL`/`MID_RAIL`/`VOL_RATIO_HOT`。 |
| **改飞书排版/字段** | 改 `build_feishu()`。终端输出在 `print_terminal()`。 |
| **改推送时点/次数** | 重跑 PowerShell 命令（见下面）。 |
| **改webhook** | 改 `FEISHU_URL` + 飞书通知配置.md。 |

**改推送时点的命令**（PowerShell）：
```powershell
$py=(Get-Command python).Source; $days='Monday','Tuesday','Wednesday','Thursday','Friday'
$times='09:45','10:30','11:15','13:30','14:15','14:50'   # 改这里

```
