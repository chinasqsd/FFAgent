# FFAgent — Personal Finance Assistant Agent Template

> **FFAgent** (全称 **FriendFinanceAgent**) 是一个个人金融投资 AI 助手模板。
> 它不是一个开箱即用的交易机器人，而是一套**投资合伙人 + 纪律闸门 + 知识库进化器**的工作流框架。

---

## 它能做什么

这个模板把一位真实独立投资者的 AI 协作体系做了通用化改造，保留了"能力"、删除了"个人状态"。拿到手后，你可以获得：

- **完整的 AI 工作规则**：AGENTS.md + 金融AI助手产品设计章程，定义了 AI 如何作为投资合伙人工作
- **交易纪律闸门**：高认知启动检查五问、错误代价分级、冷静30分钟规则、智囊团6席评审
- **行情/资金工具箱**：20个 Python 脚本，覆盖行情查询、资金流向、乖离扫盘、市场宽度、本地日K分析
- **钱脉系统框架**：盘后资金流向采集+持续性分析的完整架构文档
- **知识库蒸馏体系**：产业卡片制作规范、收件箱处理流程、个股深度调研模板
- **智囊团机制**：价值/成长/逆向/尾部风险/量化情绪/纪律审计官 6席评审章程

## 不是内置数据，是模板

此模板**不包含**：
- ❌ 任何个人持仓、资金、交易记录
- ❌ 历史复盘、经验教训、产业卡片
- ❌ 钱脉历史数据（JSON格式资金流向）
- ❌ API 密钥、webhook URL
- ❌ 调研笔记、智囊团评审记录

以上全部需要使用者自己填写和积累。

## 快速开始

```bash
# 1. 复制环境变量模板
cd FFAgent\工具
copy .env.example .env
# 编辑 .env 填入你的飞书 webhook、钱脉 API 等

# 2. 安装依赖
pip install -r requirements.txt

# 3. 填写你的持仓信息
# 编辑 持仓/股票账户持仓.md、持仓/ETF账户持仓.md 等

# 4. 测试工具
python 工具/query_stock.py 000001 399001
python 工具/lookup_stock.py 平安银行
python 工具/fund_flow.py 000001

# 5. 配置监控池
# 编辑 工具/fund_flow_monitor.py 的 STOCKS 和 STOCK_NAMES
# 编辑 工具/sector_bias_radar.py 的 GROUPS 和 SPECIAL

# 6. 启动 AI 助手（Hermes/Claude Code），将工作目录指向此项目
```

**首次使用大约需要30-45分钟**完成初始化问卷、填写持仓、配置工具。

## 技术栈

- **Python 3.10+**（标准库为主，少数脚本需要第三方包）
- 行情接口：腾讯股票 API（`qt.gtimg.cn`）
- 资金接口：同花顺 thsdk（可选，游客账户可免配）
- 通知推送：飞书 Webhook（可选）
- 前复权数据：腾讯历史 K 线接口

## 配置要求

| 配置项 | 是否必需 | 说明 |
|--------|---------|------|
| 腾讯行情API | ✅ 免费 | `query_stock.py` 等用，零配置 |
| 同花顺 thsdk | ⚠️ 可选 | `fund_flow.py` 用，游客账户可能失效 |
| 飞书 webhook | ⚠️ 可选 | 盘中乖离扫盘/资金监控推送 |
| 钱脉 API | ⚠️ 可选 | 盘后资金流向历史采集 |
| 通达信数据目录 | ⚠️ 可选 | `day_kline_analysis.py` 本地日K分析 |
